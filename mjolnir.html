Installation Guide

  1. Download the mod manager from the link above.
  2. Run the Mjolnir Mod Manager.exe
  3. Allow through virus protection if needed. (See Security Concerns for More Info)
  4. Select a Game to Mod from the List.
  5. Adjust settings in the top left settings gear if needed.
  6. Backup and or Delete Old Mods if Desired.
  7. Add Modpacks you'd like to install.
  8. Click Install Mods.

Reasons your install may have failed

  • Blocked by Virus Protection - Most Common (See Security Concerns to understand why)
    • To bypass Windows Smart Screen Click "More Info" then "Run Anyway"
  • Using and out of date version that can't connect to the newer servers. (Try downloading from the link above)

Security Concerns

Due to how most modern virus protections work this app is likely to be flagged as potentially dangerous. This is for a few reasons which are listed below.

  • It is a new app that is not well known.
  • It is a .exe file.
  • It is not signed using a +$300/year security signature (AKA Security Paywall).

That being said this app is completely safe to use and is open source so you can see for yourself using the Source Code link above.

You can even run the original python script if you want to see the code that way.
